# JMessenger CLI - Version 2.8
Copyright (c) 2026. Free use.

JMessenger è un'applicazione di messaggistica istantanea leggera basata su Java che permette comunicazioni P2P (1-a-1) e Group Chat (Multi-Unicast) all'interno di una rete locale, senza la necessità di un server centrale.

## Caratteristiche Principali

- scan             -->     Cerca altri PC nella tua rete locale che hanno JMessenger aperto.

- ip address       -->     Imposta un destinatario singolo per una chat privata 1-a-1.

- add address      --> 	   Aggiunge un indirizzo IP alla lista del gruppo.

- list             -->	   Visualizza tutti gli indirizzi IP attualmente presenti nel gruppo.

- clearlist	       -->     Svuota completamente la lista dei contatti del gruppo.

- info	           -->     Mostra il tuo IP locale, la porta e i target impostati.

- exit             -->     Chiude le connessioni e termina il programma.

- help	           -->     Mostra la lista rapida dei comandi.


## Guida rapida:

### *Conversazione P2P (2 Persone)* 

- Per avviare una conversazione bisogna scambiarsi i reciproci indirizzi IPv4 (Quello che vi viene comunicato dal comando "info" a volte puo essere errato, controllare su windows).
- Inserire l'idirizzo dell'altra persona con il comando "ip <indirizzo>".
- Scrivere liberamente sul terminale, una volta terminato "exit" chiude il programma.
